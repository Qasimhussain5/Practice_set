# Practice_set
This repository contains a collection of Python programs for beginners to practice fundamental concepts such as variables, data types, loops, conditionals, functions, and basic algorithms. These exercises help strengthen your understanding of Python programming by solving practical problems.

Topics Covered

Variables and Data Types

Input / Output

Loops (for, while)

Conditional Statements (if-else)

Functions

Basic Algorithms (Factorial, Fibonacci, Prime Numbers, GCD/LCM, etc.)

String Manipulation

Lists and Number Operations
